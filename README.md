# NHL Player Performance Dashboard (2021–2022)

## Overview
Interactive dashboard built in Google Sheets to analyze NHL player performance across teams, positions, and scoring metrics.

## Features
- Dynamic filtering by category, metric, team, and player
- Automated sorting (high-to-low / low-to-high)
- Comparative analysis across multiple players
- Pivot table breakdowns by team and position

## Tools
Google Sheets, Data Cleaning, Pivot Tables, Formulas

## Key Insights
- Identifies top-performing players by selected metrics
- Enables comparison across positions (C, LW, RW, D)
- Highlights team-level performance differences

## Example Insight
Defensemen show significantly lower goal production compared to forwards, but contribute more evenly across assists and overall point distribution depending on team structure.
